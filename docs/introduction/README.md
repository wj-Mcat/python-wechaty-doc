## 一、介绍

Python-Wechaty 是一个面向于所有IM平台的聊天机器人SDK，能够使用一套代码开发所有平台的聊天机器人，能够帮助你减少大量的工作。

目前已支持微信、微信公众号、企业微信、钉钉等平台，只需要少量的代码
